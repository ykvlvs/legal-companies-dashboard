# Дашборд юридических компаний

Статическая HTML-страница с навигацией по компаниям, рейтингам, скриншотам и ошибкам загрузки.

## Обновление данных

Из корня этого репозитория:

```bash
python3 -m http.server 8000
```

## Запуск

Из корня репозитория:

```bash
python3 -m http.server 8000
```

Затем открыть: <http://localhost:8000/>

Страница использует `data/manifest.json` и локальные копии скриншотов. Она не требует сервера приложений или базы данных.
